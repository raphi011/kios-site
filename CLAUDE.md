# Kios Marketing Site (kios.raphi011.dev)

Hugo static site for Kios, a native iOS EPUB reader. No JS framework — hand-written
layouts, one CSS file, Hugo Pipes for minify+fingerprint.

## Commands

```bash
hugo server          # dev server with live reload
hugo --minify --gc   # production build (what the Dockerfile runs)
docker build .       # full prod image: hugomods/hugo:exts → caddy:2-alpine, port 8080
```

## Architecture

- `layouts/index.html` — the **entire landing page** as one hardcoded template (no markdown)
- `layouts/_default/single.html` — **hardcoded privacy page**, not a generic template;
  `content/privacy.md` exists only to create the `/privacy/` route. New pages need either
  a new dedicated layout + stub content file, or making single.html actually generic.
- `assets/site.css` — single stylesheet; design tokens in `:root`, mirrors the app's
  kios-tokens. Dark mode is `prefers-color-scheme` only (no class toggle, no JS).
- `layouts/partials/device.html` — CSS phone bezel (`.screen-frame`) + `<picture>`
  light/dark swap. Pass `light`/`dark`/`alt`/`loading` dict params.
- `layouts/partials/custom-head.html` — Umami analytics, include in every page `<head>`.

## Screenshots (static/site-assets/)

Generated by `scripts/capture-screenshots.sh` from the iOS simulator (needs `axe`, `cwebp`,
booted sim with Kios). Captures must be opaque, 900px wide, **no baked device frame** —
the CSS bezel frames them. Light+dark pair per screen: `{home,reader,stats}-{light,dark}.webp`.

## Deploy

Push to `main` → GitHub Actions (native arm64) → Tailscale → image pushed to Zot
(`zot.manx-turtle.ts.net/kios-web`) → tag bumped in `raphi011/turingpi-k8s` (GitOps pulls it).
Caddy serves `:8080` with `/health` endpoint.

## Conventions

- Commit prefixes: `feat:`, `fix(css):`, `ci:`, `content(page):`
- Brand mark: red dot via `<span class="reddot">.</span>` (headings) and
  `<span class="dot">.</span>` (eyebrows)
- Section pattern: `.section > .wrap > .section-head` with `.eyebrow` + `.section-title`
- External links: `target="_blank" rel="noopener"`
- Key links: TestFlight `https://testflight.apple.com/join/fzdZFbtM`, Discord `https://discord.gg/spTFCkhP`
